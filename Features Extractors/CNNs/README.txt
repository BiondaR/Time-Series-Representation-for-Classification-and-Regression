Para executar a extração de features, certifique-se de executar os seguintes passos:
- Instalar todos os pacotes presentes no arquivo requirements.txt
- Executar a extração, dentro da pasta pretrainedmodels, da seguinte forma: "python extract.py path_dataset output.npy". O output.npy terá as features. 
O path_dataset é a pasta que tem as imagens. Ele vai fazer a extração em ordem alfabética.
